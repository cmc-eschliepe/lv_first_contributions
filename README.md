# lv_first_contributions
A Repository that will start as a repo for our presentation but long term serve as a place for people to test out how to make changes to and create a pull-request for open source repos.

# Fork the Repo
Please fork this Repo to your GitHub account.  If you need any help, there will be a presentation uploaded with detailed instructions for how to do that.

# Make a Change
Make a change to the Contributors.md file to include your name.

# Commit your Changes
You need to make commit the change.

# Create a Pull Request
Make a pull request to push the changes back to the main Repo.

# ReSynch your Repo
After the pull request has been accepted and merged, resynchronize your forked repo with the original upstream repo.
